# Profile 生成样本 v3

按 gen_v2.md 三步工作流生成。3 条 profile，3 种对抗类型 × 3 种意愿倾向，各取自不同 prompt。

---

## Step 1：Prompt 结构分析

### mp_04 — Wealth Seed Program（应届生储蓄+投资）

| 要素 | 内容 |
|------|------|
| 业务 | 应届毕业生专属储蓄+投资计划 |
| 产品 | Wealth Seed Program：第一年 5% 种子基金匹配、第二年起 4% 年利率、Salary Split Calculator 预算工具、$500 技能培训补贴 |
| 客服话术类型 | 身份验证 → 信息采集(毕业年份/行业) → 种子基金说明 → 预算工具介绍 → 目标设定 → 社区福利 → 激活 |
| 适用人群 | 刚毕业、有工资账户、想开始攒钱但没经验的年轻人 |
| 不适用人群 | 已有成熟理财体系、收入不稳定不想绑定自动扣款、对"匹配"概念不信任的人 |

### mp_02 — 白金卡升级权益营销

| 要素 | 内容 |
|------|------|
| 业务 | 信用卡达标客户白金升级 |
| 产品 | Platinum Card：机场接送、年度体检、高额度、免外币转换费、消费达标免年费 |
| 客服话术类型 | 身份验证 → 告知达标 → 权益介绍 → 年费说明 → 升级确认 → 渠道选择 → 配送 |
| 适用人群 | 年消费 8 万+、经常出差/旅行、有健康管理意识 |
| 不适用人群 | 消费不多、不坐飞机、对年费敏感、已有其他高端卡 |

### mp_01 — 亲子储蓄+教育金

| 要素 | 内容 |
|------|------|
| 业务 | 面向带娃家庭的储蓄产品 |
| 产品 | 方案一：亲子分期储蓄(月存 500 起，3 年期 2.5%) / 方案二：教育金理财(1 万起存，5 年期预期 3.3%+教育补贴) + 财商工作坊 + 季度报告 |
| 客服话术类型 | 身份验证(是否为家长+孩子年龄) → 储蓄痛点挖掘 → 方案介绍 → 开户链接 → 线下活动邀请 → 联系偏好确认 |
| 适用人群 | 有 10 岁以下孩子的家庭、想给孩子存教育金的家长 |
| 不适用人群 | 没有孩子 / 孩子已成年 / 家庭财务完全由配偶管理自己不做主 |
